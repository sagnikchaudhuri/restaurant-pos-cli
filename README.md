# restaurant-pos-cli
Little Lemon POS System (Node.js CLI)
A command-line restaurant POS system built with Node.js.

Features:
• Interactive ordering system
• Cart management
• Receipt generation
• Order history storage
• Revenue tracking
• Terminal UI using Chalk


## Installation

1. Clone the repository
git clone https://github.com/yourusername/little-lemon-pos-cli

2. Install dependencies
npm install

3. Run the program
node restaurant-pos.js
